Instructions:
int_ex6.sparc is the main part of the program used for the implementation. 
To run the program, enter the following commands on the command prompt:

./sparc int_ex6.sparc -o new

where "." refers to the directory of the files.

If asked to overwrite output file, enter y.

Next:

./dlv.bin -nofinitecheck -pfilter="occurs" new

Again, "." refers to the directory of the files.

If this is done correctly, the command prompt should output a subset of the answer set containing only actions.
